Importing audio data

  • tuneR allows the import of wave files as Wave objects.
  • The readAudio function of sonicscrewdriver allows reading multiple file types.
  • audio can record from system sound devices.

Linking to other tools

  • ReVAMP allows the use of Vamp Plugins for audio analysis in R.
  • Rraven allows exchanging data with Raven software and R (in particular warbleR).
  • sonicscrewdriver allows reading and creating Audacity label files (examples: Interacting with other tools)

Datasets

  • NatureSounds includes several example animal sounds.
